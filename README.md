# ES-Instruct Demo Page

Demo page for the ES-Instruct Master's thesis project at KTH Royal Institute of Technology, in collaboration with Epidemic Sound.

ES-Instruct is an instruction-tuned text-to-music editing model that takes an input audio and a natural-language prompt and performs targeted _ADD_ or _REMOVE_ edits while preserving unrelated structure.

## Notes

The demo audio files are based on music from [MoisesDB](https://music.ai/blog/press/introducing-moisesdb-the-ultimate-multitrack-dataset-for-source-separation-beyond-4-stems) and [Epidemic Sound](https://www.epidemicsound.com).

All demos are converted from WAV to MP3 for web demo loading purposes.
